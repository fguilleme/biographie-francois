# Bundles de séance

Ce dossier conserve les paquets de travail produits à la fin de certaines séances.

- Les répertoires `bundle-biographie-YYYY-MM-DD/` contiennent les fichiers décompressés.
- Les fichiers ZIP correspondants sont conservés comme copies transportables.
- `archive/` contient les transcriptions et exports historiques centralisés.

Une fois un bundle intégré, ses chapitres doivent vivre dans les dossiers narratifs principaux (`voyages/`, `professionnel/`, etc.). Le bundle reste intact pour assurer la traçabilité et ne doit pas être utilisé comme seconde source narrative active.
